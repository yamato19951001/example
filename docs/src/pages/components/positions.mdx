# Positioner

Position an element around another one. This component uses [react-popper](https://github.com/FezVrasta/react-popper) to determine positions.

#### Basic usage

```jsx
<React.Fragment>
  <Positioner
    isOpen={true}
    placement="right"
    target={({ ref }) => <Button ref={ref}>Target</Button>}
  >
    {({ placement, ref, arrowProps, style }, animation) => (
      <Layer
        ref={ref}
        data-placement={placement}
        style={{
          ...style,
          padding: "1rem",
        }}
      >
        Put me in my place
      </Layer>
    )}
  </Positioner>
</React.Fragment>
```
