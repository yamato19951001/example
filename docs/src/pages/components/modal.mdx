# Modal

A modal (or dialog) is useful for displaying infomation that commands the user's attention.

Sancho exports these components to create Modals:

- **Modal**: Your basic modal dialog which includes a ModalHeader if provided a title.
- **ModalHeader**: A standard header of a modal, which includes a title and a close button.

#### Basic usage

```jsx
<Component initialState={{ isOpen: false }}>
  {({ state, setState }) => (
    <React.Fragment>
      <Button onClick={() => setState({ isOpen: true })}>Show modal</Button>
      <Modal
        isOpen={state.isOpen}
        onRequestClose={() => setState({ isOpen: false })}
        title="Hello world!"
      >
        <div className="Modal-example">
          <Text>Hello world</Text>
        </div>
      </Modal>
    </React.Fragment>
  )}
</Component>
```

#### Fullscreen mobile example

This option enables fullscreen on mobile devices, which is typically a better user experience.

```jsx
<Component initialState={{ isOpen: false }}>
  {({ state, setState }) => (
    <React.Fragment>
      <Button onClick={() => setState({ isOpen: true })}>Show modal</Button>
      <Modal
        isOpen={state.isOpen}
        mobileFullscreen
        onRequestClose={() => setState({ isOpen: false })}
        title="Hello world!"
      >
        <div className="Modal-example">
          <Text>Hello world</Text>
        </div>
      </Modal>
    </React.Fragment>
  )}
</Component>
```

#### Without a title

When a title isn't provided, a header isn't added.

```jsx
<Component initialState={{ isOpen: false }}>
  {({ state, setState }) => (
    <React.Fragment>
      <Button onClick={() => setState({ isOpen: true })}>Show modal</Button>
      <Modal
        isOpen={state.isOpen}
        onRequestClose={() => setState({ isOpen: false })}
      >
        <div className="Modal-example">
          <Text>Hello world</Text>
          <Button onClick={() => setState({ isOpen: false })}>Cancel</Button>
        </div>
      </Modal>
    </React.Fragment>
  )}
</Component>
```
