# Modal

#### Basic usage

```jsx
<Component initialState={{ isOpen: false }}>
  {({ state, setState }) => (
    <React.Fragment>
      <Button onClick={() => setState({ isOpen: true })}>Show modal</Button>
      <Modal
        isOpen={state.isOpen}
        onRequestClose={() => setState({ isOpen: false })}
        title="Hello world!"
      >
        <div className="Modal-example">
          <Text>Hello world</Text>
        </div>
      </Modal>
    </React.Fragment>
  )}
</Component>
```

#### Fullscreen mobile example

```jsx
<Component initialState={{ isOpen: false }}>
  {({ state, setState }) => (
    <React.Fragment>
      <Button onClick={() => setState({ isOpen: true })}>Show modal</Button>
      <Modal
        isOpen={state.isOpen}
        mobileFullscreen
        onRequestClose={() => setState({ isOpen: false })}
        title="Hello world!"
      >
        <div className="Modal-example">
          <Text>Hello world</Text>
        </div>
      </Modal>
    </React.Fragment>
  )}
</Component>
```

#### Without a title

```jsx
<Component initialState={{ isOpen: false }}>
  {({ state, setState }) => (
    <React.Fragment>
      <Button onClick={() => setState({ isOpen: true })}>Show modal</Button>
      <Modal
        isOpen={state.isOpen}
        onRequestClose={() => setState({ isOpen: false })}
      >
        <div className="Modal-example">
          <Text>Hello world</Text>
          <Button onClick={() => setState({ isOpen: false })}>Cancel</Button>
        </div>
      </Modal>
    </React.Fragment>
  )}
</Component>
```
