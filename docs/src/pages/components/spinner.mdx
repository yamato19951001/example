# Spinner

Just your usual spinner component modelled after the one found in [Bootstrap](https://getbootstrap.com/docs/4.2/components/spinners/). By default it will delay for 400ms before displaying.

#### Basic usage

```jsx
<Spinner />
```

#### Other options

You can also add a label to your spinner and you can attempt to center the spinner using the `center` prop.

```jsx
<div style={{ width: "300px", height: "150px" }}>
  <Spinner label="Creating project..." center />
</div>
```
