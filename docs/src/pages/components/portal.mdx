# Portal

A portal renders a react component outside of the standard application tree. This is used by the [Overlay](/components/overlay), [Popover](/components/popover), and [Tooltip](/components/tooltip) components.

#### Basic usage

Supply the element you want rendered as the child of the portal.

```.jsx
<Portal><div>Yeah, that's simple</div></Portal>
```
