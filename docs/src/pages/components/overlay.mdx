# Overlay

An overlay used by the [Modal](/components/modal) and [Sheet](/components/sheet) components. It requests to close when clicked.

#### Basic usage

Add whatever you'd like to display over the Overlay as its children.

```jsx
<Component initialState={{ isOpen: false }}>
  {({ state, setState }) => (
    <React.Fragment>
      <Button onClick={() => setState({ isOpen: true })}>Open</Button>
      <Overlay
        isOpen={state.isOpen}
        onRequestClose={() => setState({ isOpen: false })}
      >
        <div>Click somewhere to close</div>
      </Overlay>
    </React.Fragment>
  )}
</Component>
```
