# Toast

A toast is used to notify users about a particular event. We built this with an imperative API mostly because it's way easier to invoke in response to API errors or user events. As a result, we export the lowercase `toast` function which you can use to invoke notifications. We use the library [toasted-notes](https://github.com/bmcmahen/toasted-notes) under the hood.

Toast accepts the following options:

- **title**: The title of the notification.
- **subtitle**: The subtitle of the notification.
- **duration**: How long the toast remains visible in ms. Set to null for it to remain open indefinitely.
- **position**: The position where the toast will appears. Can be top-left, top, top-right, bottom-left, bottom, bottom-right.
- **intent**: Accepts the same intent options as [Alert](/components/alert).
- **render**: Used to render custom toast notifications.

#### Basic usage

```jsx
<Button
  onClick={() => {
    toast({
      title: "Hello world!",
      duration: 3000,
      subtitle:
        "Quis anim consectetur consectetur ea. Deserunt nulla tempor est et laborum.",
    })
  }}
>
  Show toast
</Button>
```

#### Altering positions

```jsx
<Button
  onClick={() => {
    ;[
      "top-left",
      "top",
      "top-right",
      "bottom-left",
      "bottom",
      "bottom-right",
    ].map(position => {
      toast({
        title: "Hello world!",
        duration: 3000,
        position,
      })
    })
  }}
>
  Show all the toasts
</Button>
```

#### Changing the alert intent

```jsx
<Button
  onClick={() => {
    ;["info", "success", "question", "danger", "warning"].map((intent, i) => {
      setTimeout(() => {
        toast({
          title: intent,
          duration: 3000,
          intent,
          position: "top-left",
        })
      }, i * 500)
    })
  }}
>
  Show toast
</Button>
```

#### Using a custom element

You can supply a custom element by providing a render option.

```jsx
<Button
  onClick={() => {
    toast({
      render: () => <div>Hello world!</div>,
    })
  }}
>
  Show toast
</Button>
```

#### Using a render callback

Using a render callback allows you to control when the toast is closed.

```jsx
<Button
  onClick={() => {
    toast({
      render: ({ onClose, id }) => (
        <div id={id}>
          <Text>Hello world</Text>
          <Button onClick={onClose}>Close me</Button>
        </div>
      ),
    })
  }}
>
  Show toast
</Button>
```
