# Toolbar

A toolbar provides a responsive component which changes height and padding depending on the screen size. This is typically used in combination with [Navbar](/components/navbar).

#### Basic usage

Use the `compressed` prop to create a toolbar with less vertical padding.

```jsx
<Toolbar compressed>
  <Text>Lame example</Text>
</Toolbar>
```
