# VisuallyHidden

Used to maintain screen reader accessibility while hiding elements visually from users. The opposite of `aria-hidden`. For full documentation see [Reach-UI](https://ui.reach.tech/visually-hidden).

#### Basic example

```.jsx
<VisuallyHidden>Can't see me.</VisuallyHidden>
```
